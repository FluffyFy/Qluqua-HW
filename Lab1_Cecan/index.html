1.1. Ce este HTML?

HTML (HyperText Markup Language) este limbajul standard pentru crearea paginilor web.

Are un format ușor de înțeles și utilizat de către dezvoltatori.

Un exemplu de utilizare a tag-ului mark este pentru a evidenția textul important.
"Creează o lume nouă!"

Capitolul 3

3.1. Importanța HTML

HTML este fundamentul pe care se construiesc paginile web.

Fără HTML, internetul nu ar fi ceea ce este astăzi.

Este un limbaj ușor de învățat, dar puternic în utilizare.

3.2. Viitorul HTML

Pe măsură ce tehnologia evoluează, și HTML se adaptează.

Noul standard HTML5 aduce multe îmbunătățiri.

Este important să ne actualizăm cunoștințele pentru a rămâne relevanți în domeniu.
